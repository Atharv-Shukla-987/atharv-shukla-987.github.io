# 🎨 K72 Website Clone

A frontend clone of the award-winning [k72.ca](https://k72.ca) — a bold, cinematic branding agency website. Built to practice advanced UI design, smooth animations, and modern React development.

🔗 **Live Demo:** https://atharv-shukla-987.github.io/

---

## 📸 Overview

K72 is a Montreal-based branding agency known for its striking visual identity — large-scale typography, smooth scrolling, and a clean black-and-white editorial aesthetic. This project recreates that experience as a frontend clone using React and Vite.

---

## ✨ Features

- ⚡ **Fast & Lightweight** — Powered by Vite for instant dev server and optimized builds
- 🎨 **Bold Typography** — Large-scale type inspired by K72's cinematic editorial layout
- 🖱️ **Interactive UI** — Hover states and animated elements for an engaging experience
- 📱 **Responsive Design** — Works seamlessly across desktop, tablet, and mobile
- 🌍 **Deployed on GitHub Pages** — Accessible live via GitHub Pages

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Atharv-Shukla-987/atharv-shukla-987.github.io.git

# Navigate into the project
cd atharv-shukla-987.github.io

# Install dependencies
npm install

# Start the development server
npm run dev
```

Then open your browser at `http://localhost:5173`

### Build for Production

```bash
npm run build
```
